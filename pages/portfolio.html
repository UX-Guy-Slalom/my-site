SaaS Platform Redesign

Product Designer · 2024

The Problem

A B2B SaaS platform was experiencing high user churn and low feature adoption. Users found the complex interface overwhelming, and onboarding was confusing. The legacy design hadn't been updated in 3 years.

My Process

  • User Research: Conducted 12 user interviews and analyzed usage analytics from 5000+ active users
  • Insights: Found that 70% of users only used 20% of features due to poor discoverability
  • Ideation: Workshopped 8 different navigation patterns with stakeholders and tested 3 with users
  • Prototyping: Built interactive prototypes in Figma for user validation testing

The Solution

Redesigned information architecture with improved mental models. Simplified the primary navigation, created contextual onboarding, and implemented a progressive disclosure pattern that revealed advanced features only when needed.

40% Increase in user retention
3x Feature adoption rate
45% Faster task completion

Mobile Fitness App Design

Lead Product Designer · 2023

The Problem

A startup wanted to launch a fitness tracking app in a crowded market. They needed a differentiated user experience that encouraged consistent engagement and community participation while handling complex data visualization.

My Process

  • Competitive Analysis: Audited 15 existing fitness apps to identify gaps and opportunities
  • User Personas: Developed 4 detailed personas based on interviews with fitness enthusiasts
  • Iterative Design: Tested 3 different UI approaches with 50 beta users through 4 rounds of feedback
  • Gamification: Designed reward systems and social features to drive retention

The Solution

Created an engaging interface focused on progress visualization and social motivation. Implemented a activity feed, achievement badges, and friend challenges. Designed intuitive data displays that made complex workout data digestible at a glance.

95% User satisfaction (beta)
78% Weekly active users
4.8★ App store rating

E-Commerce Checkout Optimization

UX Designer · 2023

The Problem

An established e-commerce platform had a 25% cart abandonment rate during checkout. Users were experiencing friction across payment, shipping, and form validation steps, resulting in lost revenue.

My Process

  • Data Analysis: Reviewed heatmaps and session recordings to identify drop-off points
  • A/B Testing: Tested 5 different checkout flows with real users measuring conversion impact
  • Form Optimization: Reduced form fields by 40% using smart defaults and progressive profiling
  • Error Prevention: Implemented real-time validation and clear error messaging

The Solution

Redesigned the checkout into a single-page progressive flow. Eliminated unnecessary fields, provided guest checkout options, and added multiple payment methods. Enhanced error messaging and form validation to prevent mistakes before they happened.

35% Conversion rate increase
$2.3M Additional annual revenue
60% Cart abandonment reduction
